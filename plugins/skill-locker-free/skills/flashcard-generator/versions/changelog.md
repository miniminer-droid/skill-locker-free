# Changelog — Flashcard Generator

## v1.3.0
- Added Anki Import Quick-Start instructions for users who struggle with the actual import process
- Enhanced study advice with scheduling table (deck size → new cards/day → introduction timeline → daily review time)
- Added leech handling and card suspension guidance

## v1.2.0
- Added Card Density Guide with expected yields per source type (textbook page, lecture slide, article, vocab list)
- Added Reviewing Existing Cards mode for auditing user-created cards
- Added Pre-Delivery Quality Checklist (minimum information, unambiguity, recognition-only check, tags, cloze context)

## v1.1.0
- Added Language Learning Cards type (vocabulary, gender/article, conjugation, usage distinction, pronunciation)
- Added Thin Content Protocol for short source material (honest about limited yield)
- Upgraded Understanding-First Rule to a hard gate that refuses card creation and redirects

## v1.0.0
- Initial version with 5 card types, card design principles, processing pipeline, Anki-compatible output format

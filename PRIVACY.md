# Privacy Policy — Skill Locker Free

These skills are plain-text instruction files (`SKILL.md`) for Claude Code. They contain **no executable code, make no network requests, and collect, store, or transmit no data of any kind.**

- **No data collection.** The skills gather no personal information, usage data, or telemetry.
- **No network access.** They never contact external servers.
- **No tracking.** No analytics, cookies, or identifiers.

Everything these skills do happens locally inside your own Claude Code session, on text you provide. Nothing leaves your machine because of them.

The skills are MIT-licensed — read every line in this repository to verify the above for yourself.

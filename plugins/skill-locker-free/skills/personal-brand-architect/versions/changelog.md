# Changelog: Personal Brand Architect

## v1.3.0 — Iteration 3: Anti-Pattern Guards
**Diagnosis:** Skills lacked explicit negative guidance. Users could follow the framework correctly but still make domain-specific mistakes the skill didn't warn about.
**Changes:**
- Added "Common Mistakes" section with 3 specific anti-patterns
- Each mistake includes the wrong approach and the correct alternative
- Anti-patterns drawn from real-world failure modes in the skill's domain

## v1.2.0 — Iteration 2: Quality Gates and Edge Case Handling
**Diagnosis:** Framework steps lacked self-check criteria, and edge cases (solo practitioners, missing data, unusual contexts) were not addressed.
**Changes:**
- Added "Quality Gate" section with 3-4 verification checks to run before delivering output
- Added "Edge Cases" section covering atypical scenarios with specific adaptations

## v1.1.0 — Iteration 1: Conversation Flow and Worked Examples
**Diagnosis:** Skills did not explicitly instruct the model to ask for inputs before producing output. Frameworks lacked concrete good-vs-bad examples.
**Changes:**
- Added "Conversation Flow" section with explicit ask-first instructions
- Added "Quick Example" section showing strong vs weak output for the skill's primary deliverable

## v1.0.0 — Original
Initial skill with role persona, framework, output format, and principles.
